HTML 5

HTML5 memperkenalkan elemen semantik baru yang membuat struktur dokumen lebih bermakna:

  • <header>, <footer>: Untuk kepala dan kaki halaman
  • <nav>: Untuk navigasi
  • <article>: Untuk konten independen
  • <section>: Untuk pengelompokan konten
  • <aside>: Untuk konten samping/sidebar
  • <main>: Untuk konten utama
Kegunaan: Memudahkan mesin pencari dan screen reader memahami struktur halaman, meningkatkan SEO dan aksesibilitas.

HTML5 memungkinkan embed audio dan video tanpa plugin:

<video controls>
                      <source src="video.mp4" type="video/mp4">
                    </video>
                    
                    <audio controls>
                      <source src="audio.mp3" type="audio/mpeg">
                    </audio>
Kegunaan: Menggantikan Flash, mengurangi ketergantungan pada plugin eksternal, dan meningkatkan performa.

  • <canvas>: Untuk menggambar grafik 2D secara dinamis dengan JavaScript
  • SVG: Grafik vektor yang scalable
Kegunaan: Membuat game, visualisasi data, dan efek animasi langsung di browser.

HTML5 memperkenalkan tipe input baru dan atribut:

<input type="email" required>
                    <input type="date">
                    <input type="color">
                    <input type="range">
                    <input type="search">
Kegunaan: Validasi otomatis, pengalaman pengguna yang lebih baik, dan mengurangi kebutuhan JavaScript.

  • localStorage: Penyimpanan persisten
  • sessionStorage: Penyimpanan sesi
  • IndexedDB: Database berbasis objek
Kegunaan: Membuat aplikasi web yang bekerja offline (PWA), menyimpan preferensi pengguna. 

navigator.geolocation.getCurrentPosition(showPosition);
Kegunaan: Aplikasi berbasis lokasi seperti maps, rekomendasi lokasi terdekat.

Memungkinkan elemen untuk di-drag dan di-drop:

<div draggable="true" id="dragme">Drag me!</div>
Kegunaan: Antarmuka pengguna yang interaktif seperti penyusun konten.

Memungkinkan eksekusi JavaScript di background tanpa memblokir UI: 

const worker = new Worker('worker.js');
Kegunaan: Proses berat bisa dijalankan tanpa membuat halaman freeze.

Komunikasi dua arah antara client dan server:

const socket = new WebSocket('ws://example.com');
Kegunaan: Aplikasi real-time seperti chat, notifikasi langsung.

  • Atribut viewport meta tag
  • Elemen semantik yang adaptif
Kegunaan: Membuat website yang bekerja baik di semua perangkat.

HTML5 dirancang untuk bekerja sempurna dengan:

  • CSS3 untuk styling lanjutan
  • ECMAScript 6+ untuk fungsionalitas kompleks
Kegunaan: Pengembangan aplikasi web yang kaya fitur.

Microdata adalah bagian dari HTML5 yang memungkinkan Anda menyematkan metadata semantik ke dalam konten HTML Anda. 

<div itemscope itemtype="http://schema.org/Person">
                      <span itemprop="name">John Doe</span>
                      <span itemprop="jobTitle">Web Developer</span>
                    </div>
Kegunaan: Memperkaya hasil pencarian (rich snippets) dan membantu mesin pencari memahami konten.
