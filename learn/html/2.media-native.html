Media Native HTML5

Media native HTML5 adalah elemen yang digunakan untuk menampilkan konten multimedia seperti gambar, video, dan audio secara langsung di dalam halaman web tanpa memerlukan plugin tambahan.

Revolusi Media Web

HTML5 menghadirkan dukungan native untuk audio dan video tanpa memerlukan plugin eksternal seperti Flash.

Video HTML5
Contoh Implementasi:
<video width="640" height="360" controls poster="preview.jpg">
            <source src="video.mp4" type="video/mp4">
            <source src="video.webm" type="video/webm">
            Browser Anda tidak mendukung tag video.
          </video>
Atribut Penting:
  • controls : Tampilkan kontrol pemutaran
  • autoplay : Mulai otomatis (terbatas oleh browser)
  • loop : Putar ulang otomatis
  • muted : Nonaktifkan suara
  • poster : Gambar thumbnail
Format Video yang Didukung:
Format MIME Type Dukungan Browser
MP4 (H.264) video/mp4 Semua Browser
WebM (VP9) video/webm Kecuali IE
Ogg Theora video/ogg Terbatas
Audio HTML5
Contoh Implementasi:
<audio controls>
            <source src="audio.mp3" type="audio/mpeg">
            <source src="audio.ogg" type="audio/ogg">
            Browser Anda tidak mendukung tag audio.
          </audio>
Atribut Penting:
  • controls : Tampilkan kontrol pemutaran
  • autoplay : Mulai otomatis (terbatas kebijakan browser)
  • loop : Putar ulang otomatis
  • preload : Metadata/auto/none
Format Audio yang Didukung:
Format MIME Type Dukungan Browser
MP3 audio/mpeg Semua Browser
WAV audio/wav Kecuali IE
Ogg Vorbis audio/ogg Terbatas
JavaScript API untuk Media
Contoh Kontrol Media dengan JavaScript:
// Mendapatkan elemen video
          const video = document.getElementById('myVideo');
          
          // Method dasar
          video.play();  // Memutar
          video.pause(); // Menjeda
          
          // Properti penting
          video.currentTime = 30; // Lompat ke detik ke-30
          video.volume = 0.5;     // Set volume 50%
          
          // Event listener
          video.addEventListener('ended', () => {
            console.log('Video selesai diputar');
          });
Fitur API Lanjutan:
  • video.requestFullscreen() - Mode layar penuh
  • video.captureStream() - Capture untuk WebRTC
  • video.textTracks - Untuk subtitle
  • video.playbackRate - Kecepatan pemutaran

Keunggulan Media HTML5:

Performa

Lebih ringan karena native browser support tanpa plugin tambahan

Mobile Friendly

Didukung penuh di perangkat mobile tanpa Flash

Keamanan

Mengurangi risiko keamanan dari plugin eksternal

Best Practices:

  • Selalu sertakan fallback content untuk browser lama
  • Gunakan beberapa format sumber untuk kompatibilitas maksimal
  • Optimalkan video dengan compression (H.264 untuk MP4)
  • Pertimbangkan autoplay policies browser modern
  • Gunakan lazy loading untuk video di bawah halaman
