Form Upgrade

HTML5 memperkenalkan banyak fitur baru untuk formulir, membuatnya lebih interaktif dan responsif. Mari kita eksplorasi beberapa fitur baru ini.

Revolusi Input Form

HTML5 memperkenalkan 13 tipe input baru dan berbagai atribut validasi bawaan yang mengurangi ketergantungan pada JavaScript.

Tipe Input Baru
Validasi format email otomatis
Optimalkan untuk input nomor telepon di mobile
Date picker native
Nilai: 50
Contoh Kode:
<!-- Input Modern -->
          <input type="email" placeholder="Email" required>
          
          <input type="date" min="2023-01-01" max="2023-12-31">
          
          <input type="range" min="0" max="100" step="5">
Validasi Bawaan
Wajib diisi
Harus 3 huruf (regex)
Min 3, max 10 karakter
Usia 18-99 tahun
Custom Validation Message: 
// JavaScript custom validation
          const input = document.getElementById('myInput');
          input.addEventListener('invalid', () => {
            if(input.validity.valueMissing) {
              input.setCustomValidity('Wajib diisi!');
            } else if(input.validity.patternMismatch) {
              input.setCustomValidity('Format tidak sesuai');
            }
          });
          
          input.addEventListener('input', () => {
            input.setCustomValidity('');
          });

Form API & JavaScript
Constraint Validation API:
  • checkValidity() - Cek validitas form
  • reportValidity() - Tampilkan pesan error
  • validity - Objek berisi status validasi
  • validationMessage - Pesan error default
Fitur Modern:
  • FormData API - Untuk AJAX file upload
  • Autofocus - Fokus otomatis ke field
  • Spellcheck - Koreksi ejaan
  • Form Association - Element form di luar <form>
Contoh Penggunaan FormData: 
// Mengambil data form
          const form = document.getElementById('myForm');
          const formData = new FormData(form);
          
          // Mengirim via Fetch API
          fetch('/submit', {
            method: 'POST',
            body: formData
          })
          .then(response => response.json())
          .then(data => console.log(data));

Best Practices:

  • Selalu gunakan <label> yang terkait dengan for dan id
  • Tambahkan placeholder sebagai contoh input
  • Grupkan field terkait dengan <fieldset> dan <legend>
  • Validasi sisi server tetap diperlukan
  • Gunakan autocomplete untuk pengalaman pengguna lebih baik
