Web Responsive

Web Responsive adalah pendekatan desain web yang memastikan tampilan dan fungsionalitas yang optimal di berbagai perangkat dan ukuran layar.

Desain Web Modern

HTML5 menyediakan berbagai fitur untuk membuat website yang responsif dan optimal di berbagai perangkat.

Viewport Meta Tag
Dasar Penggunaan:
<meta name="viewport" 
                content="width=device-width, 
                         initial-scale=1,
                         maximum-scale=1,
                         user-scalable=no">
Parameter Penting:
  • width=device-width - Sesuaikan dengan lebar perangkat
  • initial-scale=1 - Zoom awal 100%
  • maximum-scale=1 - Batasi zoom (tidak selalu disarankan)
  • user-scalable=no - Nonaktifkan zoom (hati-hati aksesibilitas)

CSS Media Queries
Breakpoint Umum:
/* Mobile (default) */
          body { font-size: 14px; }
          
          /* Tablet */
          @media (min-width: 768px) {
            body { font-size: 16px; }
          }
          
          /* Desktop */
          @media (min-width: 992px) {
            body { font-size: 18px; }
          }
          
          /* Large Desktop */
          @media (min-width: 1200px) {
            body { font-size: 20px; }
          }
Jenis Media Query:
  • min-width - Lebar minimum
  • max-width - Lebar maksimum
  • orientation - Portrait/landscape
  • resolution - Kepadatan piksel
  • hover - Dukungan hover
Mobile-First vs Desktop-First:

Mobile-First (disarankan): Mulai styling untuk mobile lalu tambahkan untuk layar lebih besar dengan min-width.

Gambar Responsif
Teknik Dasar:
<!-- Gambar fleksibel -->
          <img src="image.jpg" alt="Deskripsi" 
               class="img-fluid">
          
          <!-- Gambar berbeda untuk resolusi berbeda -->
          <picture>
            <source media="(min-width: 1200px)" 
                    srcset="large.jpg">
            <source media="(min-width: 768px)" 
                    srcset="medium.jpg">
            <img src="small.jpg" alt="Deskripsi" 
                 class="img-fluid">
          </picture>
          
          <!-- Gambar dengan density berbeda -->
          <img src="image.jpg" 
               srcset="image@1x.jpg 1x, 
                       image@2x.jpg 2x"
               alt="Deskripsi">

  • sm: ≥576px
  • md: ≥768px
  • lg: ≥992px

Best Practices:

  • Gunakan pendekatan mobile-first
  • Uji di berbagai perangkat nyata (emulator tidak cukup)
  • Optimalkan performa untuk jaringan mobile
  • Gunakan touch-friendly elements (minimal 48x48px)
  • Hindari pop-up yang mengganggu di mobile
  • Optimalkan typography untuk keterbacaan
