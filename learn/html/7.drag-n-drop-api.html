Drag And Drop API

HTML5 memperkenalkan Drag and Drop API yang memungkinkan elemen untuk di-drag (diseret) dan di-drop (dilepas) dengan event JavaScript native. API ini sangat berguna untuk interaksi pengguna yang lebih baik, seperti mengatur ulang elemen, mengunggah file, dan membuat antarmuka yang lebih interaktif.

Interaksi Drag and Drop

HTML5 Drag and Drop API memungkinkan elemen untuk di-drag (diseret) dan di-drop (dilepas) dengan event JavaScript native.

Dasar Penggunaan
Langkah Implementasi:
  1. Set elemen sebagai draggable="true"
  2. Tambahkan event listener dragstart
  3. Tentukan zona drop dengan dragover dan drop
  4. Atur transfer data dengan DataTransfer
Event Penting:
  • dragstart: Saat mulai drag
  • dragend: Saat drag berakhir
  • dragover: Saat elemen di atas zona drop
  • drop: Saat elemen di-drop
Contoh Kode Dasar:
<!-- Elemen yang bisa di-drag -->
          <div id="dragItem" draggable="true">Drag me!</div>
          
          <!-- Zona drop -->
          <div id="dropZone">Drop here</div>
          
          <script>
            const dragItem = document.getElementById('dragItem');
            const dropZone = document.getElementById('dropZone');
          
            // Saat mulai drag
            dragItem.addEventListener('dragstart', (e) => {
              e.dataTransfer.setData('text/plain', dragItem.id);
            });
          
            // Izinkan drop
            dropZone.addEventListener('dragover', (e) => {
              e.preventDefault();
            });
          
            // Handle drop
            dropZone.addEventListener('drop', (e) => {
              e.preventDefault();
              const id = e.dataTransfer.getData('text/plain');
              const draggedItem = document.getElementById(id);
              dropZone.appendChild(draggedItem);
            });
          </script>

DataTransfer Object
Metode Utama:
  • setData(format, data) - Menyimpan data
  • getData(format) - Mengambil data
  • clearData() - Menghapus data
  • setDragImage(element, x, y) - Custom drag image
Format Data:
  • text/plain
  • text/html
  • text/uri-list
  • application/json
Contoh Transfer Data Kompleks:
// Saat dragstart
          dragItem.addEventListener('dragstart', (e) => {
            // Transfer berbagai jenis data
            e.dataTransfer.setData('text/plain', 'Ini teks biasa');
            e.dataTransfer.setData('text/html', '<strong>Ini HTML</strong>');
            e.dataTransfer.setData('application/json', JSON.stringify({id: 123, name: 'Item'}));
            
            // Set custom drag image
            const dragIcon = document.createElement('div');
            dragIcon.textContent = '📦';
            document.body.appendChild(dragIcon);
            e.dataTransfer.setDragImage(dragIcon, 10, 10);
            setTimeout(() => document.body.removeChild(dragIcon), 0);
          });
          
          // Saat drop
          dropZone.addEventListener('drop', (e) => {
            const plainText = e.dataTransfer.getData('text/plain');
            const html = e.dataTransfer.getData('text/html');
            const json = JSON.parse(e.dataTransfer.getData('application/json'));
          });

Best Practices:

  • Selalu berikan visual feedback saat dragging (ubah cursor, opacity, dll)
  • Gunakan preventDefault() pada event dragover dan drop
  • Untuk kompleksitas tinggi, pertimbangkan library seperti SortableJS atau Draggable
  • Uji di berbagai browser karena implementasi bisa berbeda
  • Pada mobile, tambahkan touch event untuk pengalaman yang lebih baik
Library Drag and Drop Populer
SortableJS

Library untuk membuat list yang bisa di-sort dengan drag and drop. 

new Sortable(listEl, {
            animation: 150,
            ghostClass: 'sortable-ghost'
          });
Draggable

Library modular untuk fungsionalitas drag and drop yang kompleks. 

new Draggable(el, {
            plugins: [SwapPlugin]
          });
React DnD

Untuk aplikasi React dengan drag and drop.

const [{isDragging}, drag] = useDrag({
            type: 'BOX',
            item: { id }
          });
