Grafik dan Animasi

HTML5 memperkenalkan banyak fitur baru untuk grafik dan animasi, termasuk elemen <canvas>, dukungan untuk SVG, dan API untuk multimedia.

Revolusi Grafik Web

HTML5 memperkenalkan dua teknologi utama untuk grafik dan animasi: <canvas> untuk grafik bitmap dan SVG untuk grafik vektor.

Canvas HTML5
Dasar Penggunaan Canvas:
<canvas id="myCanvas" width="400" height="200">
            Browser Anda tidak mendukung canvas.
          </canvas>
          
          <script>
            const canvas = document.getElementById('myCanvas');
            const ctx = canvas.getContext('2d');
            
            // Gambar persegi
            ctx.fillStyle = 'red';
            ctx.fillRect(10, 10, 100, 100);
          </script>
Fitur Utama Canvas:
  • 2D Drawing - Garis, bentuk, teks
  • Image Manipulation - Memproses gambar pixel
  • Animasi - Dengan requestAnimationFrame
  • Data Visualization - Chart, diagram
Contoh Animasi Canvas: 
// Animasi bola memantul
          function animate() {
            ctx.clearRect(0, 0, canvas.width, canvas.height);
            
            // Gambar bola
            ctx.beginPath();
            ctx.arc(x, y, 20, 0, Math.PI*2);
            ctx.fillStyle = 'blue';
            ctx.fill();
            
            // Update posisi
            x += dx;
            y += dy;
            
            // Deteksi tumbukan
            if(x + dx > canvas.width-20 || x + dx < 20) dx = -dx;
            if(y + dy > canvas.height-20 || y + dy < 20) dy = -dy;
            
            requestAnimationFrame(animate);
          }
          animate();
Scalable Vector Graphics (SVG)
Contoh SVG Langsung:
Kode SVG:
<svg width="400" height="200">
            <rect x="10" y="10" width="100" height="100" 
                  fill="red" />
            
            <circle cx="200" cy="60" r="50" 
                    fill="green" />
            
            <text x="300" y="70" 
                  font-family="Arial" 
                  font-size="20" 
                  fill="blue">
              Hello SVG!
            </text>
          </svg>
Perbandingan Canvas vs SVG:
Aspek Canvas SVG
Tipe Grafik Bitmap (pixel-based) Vektor (XML-based)
Performansi Bagus untuk kompleksitas tinggi Lebih lambat untuk banyak elemen
Interaktivitas Perlu tracking manual Event handler per elemen
Skalabilitas Bisa pecah saat zoom Tetap tajam di segala ukuran
Aplikasi Praktis
Game

Canvas ideal untuk game 2D dengan banyak objek dan animasi kompleks.

Data Visualization

SVG cocok untuk chart interaktif yang perlu scaling dan tooltips.

Gambar Vektor

SVG sempurna untuk logo, ikon, dan ilustrasi yang perlu responsif.

Library Populer
Untuk Canvas:
  • Fabric.js - Library canvas yang powerful untuk manipulasi objek
  • Konva.js - Framework 2D canvas dengan dukungan layer dan event
  • Chart.js - Library pembuatan chart berbasis canvas
Untuk SVG:
  • Snap.svg - Library manipulasi SVG modern
  • D3.js - Library visualisasi data berbasis SVG
  • SVG.js - Library ringan untuk bekerja dengan SVG

Best Practices:

  • Gunakan Canvas untuk animasi kompleks dan game
  • Pilih SVG untuk ilustrasi dan UI yang perlu scaling
  • Optimalkan animasi dengan requestAnimationFrame
  • Untuk grafik kompleks, pertimbangkan WebGL (3D)
  • Gunakan CSS transform untuk animasi performa tinggi
