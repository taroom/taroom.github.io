# Modules dan crate structure

Modules membantu mengatur kode jadi rapi dan mudah dirawat.

## Struktur sederhana

```text
src/
  main.rs
  auth/
    mod.rs
    service.rs
```

## Contoh deklarasi module

```rust
mod auth;

fn main() {
    println!("app started");
}
```

Di `src/auth/mod.rs`:

```rust
pub mod service;
```

Di `src/auth/service.rs`:

```rust
pub fn login() {
    println!("login");
}
```

## Rekomendasi

- Gunakan visibilitas `pub` seperlunya.
- Pisahkan domain per folder agar struktur mudah diikuti.
