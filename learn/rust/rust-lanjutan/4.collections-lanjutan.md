# Collections lanjutan

Rust menyediakan koleksi penting di standard library.

## Vector

```rust
let mut nums = vec![1, 2, 3];
nums.push(4);
```

## HashMap

```rust
use std::collections::HashMap;

let mut scores = HashMap::new();
scores.insert("taro", 90);
scores.insert("rani", 88);
```

## String

```rust
let mut name = String::from("Rust");
name.push_str(" Lang");
```

## Tips performa

- Pilih struktur data sesuai use case.
- Hindari clone berlebihan untuk data besar.
