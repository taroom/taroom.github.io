# Async Rust dengan Tokio

Untuk I/O intensif seperti API dan networking, async sangat membantu.

## Tambahkan dependency

```toml
[dependencies]
tokio = { version = "1", features = ["full"] }
```

## Contoh async main

```rust
#[tokio::main]
async fn main() {
    let result = fetch_data().await;
    println!("{}", result);
}

async fn fetch_data() -> String {
    String::from("done")
}
```

## Kapan pakai async

- Banyak operasi I/O bersamaan.
- Server dengan banyak koneksi.

Jika workload CPU-heavy, kombinasikan async dengan task atau thread pool yang tepat.
