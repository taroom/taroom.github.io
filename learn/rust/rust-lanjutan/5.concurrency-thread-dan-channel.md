# Concurrency dengan thread dan channel

Rust punya dukungan concurrency aman lewat ownership dan type system.

## Thread dasar

```rust
use std::thread;

fn main() {
    let handle = thread::spawn(|| {
        for i in 1..=3 {
            println!("thread: {}", i);
        }
    });

    handle.join().unwrap();
}
```

## Channel untuk komunikasi

```rust
use std::sync::mpsc;
use std::thread;

fn main() {
    let (tx, rx) = mpsc::channel();

    thread::spawn(move || {
        tx.send(String::from("hello from thread")).unwrap();
    });

    let msg = rx.recv().unwrap();
    println!("msg: {}", msg);
}
```

Gunakan channel untuk komunikasi antar thread tanpa shared mutable state langsung.
