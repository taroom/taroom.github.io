# Variabel, tipe data, dan kontrol alur

## Variabel dan mutability

```rust
fn main() {
    let x = 10;
    let mut y = 20;

    y += 5;
    println!("x: {}, y: {}", x, y);
}
```

## Tipe data umum

- Integer: `i32`, `i64`, `u32`
- Floating point: `f32`, `f64`
- Boolean: `bool`
- Character: `char`
- String: `String` dan `&str`

## Kontrol alur

```rust
fn main() {
    let n = 7;

    if n % 2 == 0 {
        println!("genap");
    } else {
        println!("ganjil");
    }

    for i in 1..=3 {
        println!("{}", i);
    }
}
```
