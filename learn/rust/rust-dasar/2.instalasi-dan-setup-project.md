# Instalasi dan setup project

## Instalasi Rust

Windows/macOS/Linux bisa pakai rustup:

```bash
rustup-init
```

Setelah selesai, cek versi:

```bash
rustc --version
cargo --version
```

## Membuat project baru

```bash
cargo new hello_rust
cd hello_rust
cargo run
```

## Struktur project default

```text
hello_rust/
  Cargo.toml
  src/
    main.rs
```

- `Cargo.toml`: konfigurasi project dan dependency.
- `src/main.rs`: entry point aplikasi.
