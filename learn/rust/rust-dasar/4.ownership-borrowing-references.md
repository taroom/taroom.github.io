# Ownership, borrowing, dan references

Konsep ownership adalah inti Rust.

## Aturan ownership

1. Setiap value punya satu owner.
2. Satu waktu hanya ada satu owner aktif.
3. Saat owner keluar scope, value di-drop.

## Contoh move

```rust
fn main() {
    let s1 = String::from("hello");
    let s2 = s1;
    // s1 tidak bisa dipakai lagi
    println!("{}", s2);
}
```

## Contoh borrowing

```rust
fn print_len(text: &String) {
    println!("panjang: {}", text.len());
}

fn main() {
    let s = String::from("rust");
    print_len(&s);
    println!("{}", s);
}
```

## Mutable reference

```rust
fn add_mark(text: &mut String) {
    text.push('!');
}
```

Dalam satu scope, Rust membatasi kombinasi mutable dan immutable reference agar aman dari data race.
