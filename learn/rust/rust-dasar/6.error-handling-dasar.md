# Error handling dasar

Rust punya dua pola utama:

- `Result<T, E>` untuk error yang bisa diprediksi.
- `panic!` untuk kondisi fatal.

## Contoh Result

```rust
use std::fs;

fn main() {
    let result = fs::read_to_string("config.txt");

    match result {
        Ok(content) => println!("isi file: {}", content),
        Err(err) => println!("gagal baca file: {}", err),
    }
}
```

## Operator `?`

```rust
use std::fs;
use std::io;

fn load_config() -> Result<String, io::Error> {
    let content = fs::read_to_string("config.txt")?;
    Ok(content)
}
```

Gunakan `?` untuk propagasi error secara ringkas dan bersih.
