# Pengenalan Rust

Rust adalah bahasa pemrograman system-level yang fokus pada:

- Performa tinggi
- Keamanan memory tanpa garbage collector
- Concurrency yang lebih aman

## Kapan Rust cocok dipakai

- Aplikasi yang butuh performa tinggi.
- Service backend yang stabil dan efisien.
- Tool CLI, game engine component, dan software infrastruktur.

## Kelebihan Rust

- Compiler sangat ketat sehingga banyak bug tertangkap lebih awal.
- Ownership model membantu menghindari memory leak dan data race.
- Ekosistem package dikelola lewat Cargo.
