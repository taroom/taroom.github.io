# Struct, enum, dan pattern matching

## Struct

```rust
struct User {
    id: u64,
    name: String,
}

fn main() {
    let user = User {
        id: 1,
        name: String::from("Taro"),
    };

    println!("{} - {}", user.id, user.name);
}
```

## Enum

```rust
enum Status {
    Active,
    Inactive,
}
```

## Pattern matching dengan match

```rust
fn message(status: Status) -> &'static str {
    match status {
        Status::Active => "akun aktif",
        Status::Inactive => "akun nonaktif",
    }
}
```

`match` membantu menangani semua kemungkinan nilai secara eksplisit.
