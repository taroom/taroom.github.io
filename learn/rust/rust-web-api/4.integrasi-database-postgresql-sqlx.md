# Integrasi database PostgreSQL dengan sqlx

## Dependency tambahan

```toml
sqlx = { version = "0.8", features = ["runtime-tokio-rustls", "postgres", "macros"] }
dotenvy = "0.15"
```

## Koneksi pool

```rust
use sqlx::postgres::PgPoolOptions;

let database_url = std::env::var("DATABASE_URL").unwrap();
let pool = PgPoolOptions::new()
    .max_connections(5)
    .connect(&database_url)
    .await
    .unwrap();
```

## Query sederhana

```rust
let rows = sqlx::query!("SELECT id, name FROM users ORDER BY id DESC")
    .fetch_all(&pool)
    .await
    .unwrap();
```

## Saran

- Pisahkan query ke repository module.
- Pakai migration tool sqlx atau refinery untuk versioning schema.
