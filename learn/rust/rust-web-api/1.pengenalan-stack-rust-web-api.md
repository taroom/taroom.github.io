# Pengenalan stack Rust Web API

Untuk membangun Web API di Rust, stack umum yang stabil dan populer:

- Framework HTTP: Axum
- Runtime async: Tokio
- Database access: sqlx
- Serialization JSON: serde
- Config env: dotenvy

## Kenapa Axum

- Ringan dan modern.
- Type-safe karena memanfaatkan sistem tipe Rust.
- Integrasi bagus dengan Tower middleware.

## Alur umum project API

1. Definisikan route dan handler.
2. Buat layer service/repository.
3. Integrasi database.
4. Tambahkan middleware dan auth.
5. Logging, observability, dan deployment.
