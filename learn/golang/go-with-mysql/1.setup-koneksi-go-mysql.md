# Setup koneksi Go dan MySQL

## Persiapan

- Pastikan MySQL berjalan.
- Buat database, contoh: `belajar_go`.
- Siapkan user database yang punya akses.

## Install driver MySQL

```bash
go get github.com/go-sql-driver/mysql
```

## Contoh koneksi dasar

```go
package main

import (
    "database/sql"
    "fmt"
    _ "github.com/go-sql-driver/mysql"
)

func main() {
    dsn := "root:password@tcp(127.0.0.1:3306)/belajar_go?parseTime=true"

    db, err := sql.Open("mysql", dsn)
    if err != nil {
        panic(err)
    }
    defer db.Close()

    if err := db.Ping(); err != nil {
        panic(err)
    }

    fmt.Println("koneksi mysql berhasil")
}
```

## Catatan

- Gunakan environment variable untuk DSN.
- Jangan hardcode username dan password di source code.
