# CRUD sederhana

Contoh query CRUD menggunakan package database/sql.

## Create

```go
result, err := db.Exec("INSERT INTO users(name, email) VALUES(?, ?)", name, email)
id, _ := result.LastInsertId()
```

## Read

```go
rows, err := db.Query("SELECT id, name, email FROM users")
defer rows.Close()

for rows.Next() {
    var u User
    _ = rows.Scan(&u.ID, &u.Name, &u.Email)
}
```

## Update

```go
_, err := db.Exec("UPDATE users SET name = ?, email = ? WHERE id = ?", name, email, id)
```

## Delete

```go
_, err := db.Exec("DELETE FROM users WHERE id = ?", id)
```

## Tips

- Selalu cek error dari Exec, Query, dan Scan.
- Gunakan prepared statement untuk query yang sering dipakai.
